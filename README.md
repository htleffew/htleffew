# Heather Leffew, PhD

AI evaluation researcher and data scientist building measurement systems for generative AI in interactive entertainment. My work spans empirical LLM behavioral analysis, formal safety frameworks, and production evaluation harnesses for AI-driven gameplay.

## Current Work

**Applied evaluation** — I built a multi-layer evaluation harness for [Dead Signal](https://dead-signal.web.app), an AI-driven detective game running live Gemini dialogue generation. The harness includes seven deterministic evaluators, LLM-as-a-Judge scoring, golden-set regression suites, and integrated red-team probes — all operating on every model turn in production.

**Empirical LLM research** — I study unsanctioned behavioral patterns in frontier language models through large-scale corpus analysis. My [LCR analysis](https://github.com/htleffew/claude-lcr-analysis) characterized pathologizing behavior in Claude Sonnet 4.5 across 26,158 Reddit observations with a five-property structural signature (100% unsolicited, 0% yielded to pushback). Companion [sleep-nudge study](https://github.com/htleffew/claude-sleep-analysis) in progress.

**Formal safety** — I develop context-conditioned safety constraint frameworks using constrained Markov decision processes, addressing the failure mode of static alignment thresholds in dynamic interaction environments.
